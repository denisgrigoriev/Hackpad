# Minimalistic Hackpad
In this project i would be design a ultra small and minimal HackPad, this removes unnessary keys and only provides keys would would be used often, for example one key could be to copy, another paste and another could be used to screenshot and and so forth.

## Goals

- Each key would be useful.
- This project focus more on aesthetic rather than functionality.
- The whole Hackpad could fit in your hand.
